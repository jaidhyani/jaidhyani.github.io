Terminal Setup

iTerm2 + tmux + zsh on macOS, looking good out of the box

bash <(curl -fsSL https://jai.one/terminal-setup/setup.sh) click to copy

What you get

  • iTerm2 with quake-mode dropdown (system-wide hotkey)
  • tmux Catppuccin Mocha theme with rounded pill status bar
  • Live CPU sparklines, RAM meter, and network throughput in status bar
  • Powerlevel10k zsh prompt with Nerd Font icons
  • fzf fuzzy finder everywhere, including tmux pane search

tmux keybindings

Ctrl+A | Split vertically Ctrl+A - Split horizontally Ctrl+A f Fuzzy search windows/panes Ctrl+A / Search scrollback Ctrl+A c New window

How it works

  • Interactive setup — confirms before every action
  • Installs via Homebrew (tmux, fzf, powerlevel10k, Nerd Font)
  • Writes configs to ~/.tmux.conf and scripts to ~/bin/
  • Quake mode is a manual iTerm2 setting (script walks you through it)
  • Safe to re-run — checks for existing files

Works on macOS and Linux. System monitor adapts automatically.
