The headline contrast

Side by side: without Luthien vs with Luthien 8s

Same prompt, "How do I install the requests library?". Direct Claude says pip install requests; routed through Luthien with a pip→uv rewrite policy active, the response comes back as uv pip install requests.

Install

One-line install 13s

Using Claude Code through the proxy

Without Luthien 8s

claude -p direct, no proxy. Returns pip install requests.

With Luthien 8s

Same prompt via luthien claude -p; pip→uv rewrite policy active. Returns uv pip install requests.

Full Claude Code TUI 7s

Not -p: the actual interactive Ink TUI through the proxy. Prompt typed, response streams back rewritten in place.

Configuring policies

CLI tour 6s

Web UI 11s

The audit trail

Conversation history 17s

Every session and every turn recorded. Click into a conversation; each turn is tagged POLICY MODIFIED with an explicit string_replacement: Response modified by string replacement marker. The proxy is a control plane — and a black-box recorder.
